# Ohm Law Calculator

Простое приложение на C# Windows Forms для расчёта силы тока по закону Ома.

## Функциональность

- Ввод напряжения (U) через NumericUpDown
- Ввод сопротивления (R) через NumericUpDown
- Расчёт силы тока по формуле: I = U / R
- Вывод результата в текстовое поле

## Интерфейс

- numericUpDown1 — напряжение (В)
- numericUpDown2 — сопротивление (Ом)
- textBox1 — сила тока (А)

## Пример работы

U = 220 В, R = 10 Ом
Результат: 22 А

## Куда можно встроить

- В электротехнические калькуляторы
- В обучающие приложения по физике
- В инженерные расчёты
- В демонстрацию работы NumericUpDown

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При R = 0 деление невозможно — программа вызовет исключение. Для production рекомендуется добавить проверку на ноль.
